Generating a Music Sheet Page

  • Specify the Number of Clefs

    To generate a music sheet page, you must first specify the number of clefs your sheet music will have. A normal piano sheet music may have either 2 or 3 clefs. Why? Well any dynamic markings placed in the middle between a treble and bass cleff also counts as a clef! So for those that wish to be expressive, remember to set clefs as 3. The syntax for this command is "\clefnum{in}", where in=3 if you want 3 clefs. Input may only be a positive number.

  • Adding Measures

    After that, we may begin creating measures by adding notes and rests to each measure. The command to create a measure is "\measure{cleffnum=in1,clef=in2,keysig=in3,timesig=in4,notes}"

    in1 is which clef you would like to create the new measure on, indexed starting from 0 and must be a positive integer.

    in2 specifies to display a clef if you wish to, only acceptable inputs are 'treble', 'bass', 'tenor', and 'alto'. (Optional argument)

    in3 is the keysignature if you also wish to display on at the start of a measure, acctable inputs are 'a','b','c','d','e','f','g','A','B','C','D','E','F','G', 'Ab','Bb','Cb','Db','Eb','Fb','Gb',' A#','B#','C#','D#','E#','F#','G#','a#','b#','c#','d#','e#','f#','g#'.(Optional argument)

    in4 specifies a time signature. Acceptable signatures can be is any number/any number. For example "4/4" is standard time.(Optional argument)

    notes will be explained in next section. Notes can be any number of notes, as long as they are note types.

  • Dynamics

    "\crescendo{i1}" for a cresencdo of length i1, a positive integer.

    "\decrescendo{i1}" for a decresencdo of length i1, a positive integer.

    "\text{i1}" for any custom dynamic markings text i1. Thats all! Have fun!
